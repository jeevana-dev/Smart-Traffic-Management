Emergency Controls
Critical System:

Emergency mode has the HIGHEST priority and overrides all other modes. When active, the signal cycles continuously through all colors in rapid succession to alert all vehicles. Only manual resolution can deactivate this mode.

Emergency Mode Details
Mode Type: Emergency / Accident
Priority Level: HIGHEST (1st tier)
Signal Behavior: Rapid cycling + Alert
Override Capability: Overrides ALL modes
Deactivation: Manual resolution only
Expected Signal Behavior

When Emergency Mode is activated:

  1. All other modes are immediately suspended
  2. Signal rapidly cycles through RED → YELLOW → GREEN
  3. Emergency alert banner displays prominently
  4. All timers are overridden
  5. System waits for manual resolution
  6. After resolution, returns to previous mode
